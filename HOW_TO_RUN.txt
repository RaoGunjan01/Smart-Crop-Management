WHAT IS THIS PROJECT?
This is a computer simulation that acts like a smart farm manager for watering crops. It uses weather forecasts and soil sensors to decide exactly how much water each field needs so you don't waste water or let the crops get thirsty. It's like having a digital brain that watches your fields 24/7 to make sure every drop of water is used perfectly.

WHAT DO ALL THESE FILES AND FOLDERS DO?
- irrigation_env/ — This is the core folder that holds all the "rules" for how soil and plants behave.
- env.py — This the main brain of the simulation that connects everything together.
- simulator.py — This file simulates the real world, like the sun, the rain, and how water soaks into the dirt.
- tasks.py — This is where you can find different settings for simple or complex farm setups.
- reward.py — This calculates a "score" to tell the computer if it's doing a good job or a bad job.
- grader.py — This looks at a whole season of farming and gives you a final grade from 0 to 1.
- api/main.py — This is a "connector" that lets the web dashboard talk to the simulation brain.
- agents/ — This folder contains the different AI drivers that can move the simulation.
- baseline_agent.py — A simple AI that follows basic "if-then" rules like a human would.
- rl_agent.py — A high-tech AI that learns by practicing millions of times to find the best strategy.
- ui/dashboard.html — The visual control panel where you can watch the farm and click buttons.
- tests/ — A set of automated checks to make sure the computer isn't making mistakes.
- Dockerfile — A shortcut for experts to run this in a special "container" box.
- requirements.txt — A list of extra tools the computer needs to download to run this project.
- README.md — A technical manual for people who already know how to code.

HOW TO INSTALL EVERYTHING (do this once)
1. Open a terminal window.
2. Go to this project folder.
3. Create and activate a virtual environment:
   python3 -m venv .venv
   source .venv/bin/activate
4. Install dependencies:
   python -m pip install -U pip
   pip install -r requirements.txt

NOTE (Linux / CPU-only): Installing Stable-Baselines3 will pull PyTorch. If you don't need GPU/CUDA, you can install a CPU-only PyTorch build first (then install the rest):
   pip install --index-url https://download.pytorch.org/whl/cpu torch
   pip install -r requirements.txt

HOW TO START THE SYSTEM
1. In your terminal, type this to start the simulation brain:
   python3 -m uvicorn api.main:app --host 0.0.0.0 --port 8000
2. Leave this window open! It needs to stay running while you play.

HOW TO OPEN THE DASHBOARD
1. Open your web browser (like Chrome or Safari).
2. Use "File > Open" and select the file "ui/dashboard.html" from this project folder.
- Scenario Setup (recommended):
  - On load, a scenario pop-up opens.
  - Choose Crop, Season, Soil Moisture Level, Nutrient Level, and Land Area (ha), then click "Start Simulation".
- RESET: Starts a brand new season (resets charts/logs and reloads the environment).
- MANUAL STEP: Waters the fields based on the Zone controls (OFF/LOW/MED/HIGH).
- AUTO STEP (AI): Lets the AI pick an action for one time-step and logs a natural-language explanation.
- AUTO RUN LOOP: Repeats "AUTO STEP (AI)" until the episode ends (or until you click again to stop).
- Minimal UI (new):
  - Click "Minimal UI" (top-right).
  - It shows a simplified simulation view with:
    - a decision log written in English (day-by-day),
    - reward/punish counters,
    - and a profit-style summary (water used/saved and estimated money saved), scaled by the land area you selected.

HOW TO CHANGE THE DIFFICULTY
- Easy: One single field with perfect sensors and plenty of water.
- Medium: Four fields with slightly fuzzy sensors and less water.
- Hard: Eight fields with sensors that sometimes break and very little water to spare.

HOW TO TWEAK THE SIMULATION
1. To give yourself more water: Open irrigation_env/tasks.py, find "water_budget_liters=10000", and change the number.
2. To make it rain more often: Open irrigation_env/tasks.py, find "rain_probability=0.3", and change it to 0.8.
3. To make the field bigger or smaller: Open irrigation_env/tasks.py, find "n_zones=4", and change the number.
4. To make the season longer: Open irrigation_env/tasks.py, find "n_days=60", and change it to 120.
5. To change the water cost: Open irrigation_env/env.py, find "cost_per_liter=0.005", and change the price.

HOW TO TRAIN THE AI AGENT
1. Open a new terminal window.
2. Type this command and press Enter:
   python3 -m agents.rl_agent
3. The screen will fill with numbers while the AI "practices" watering the fields for a few minutes.

WHAT THE SCORE MEANS
- 1.0 (Perfect): You kept all plants healthy, used almost no water, and finished the whole season.
- 0.7 (Good): A solid job, but maybe you wasted some water or let the soil get a bit too dry.
- 0.4 (Poor): You might have run out of water early or let the plants get very stressed.
- 0.0 (Failed): The plants died or you ignored the fields entirely.

IF SOMETHING BREAKS
1. "Command not found": Make sure you installed everything using the "pip3" command above.
2. "Address already in use": Close the terminal and try again, or restart your computer.
3. "Dashboard is blank": Make sure the terminal with "api/main.py" is still running in the background.
4. "Connection Refused": Your computer's security might be blocking the dashboard. Try a different browser.
5. "Simulation looks frozen": Try clicking the "RESET" button on the left of the dashboard.
